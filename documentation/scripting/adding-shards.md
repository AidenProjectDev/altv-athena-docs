# Adding a Shard

Shard(s) are large text displays that are displayed in the center of the screen.

![](https://i.imgur.com/iUMMpkd.jpeg)

# Usage

```typescript
playerFuncs.emit.createShard(player, {
    duration: 5000,
    title: '~r~Hello World!',
    text: 'Shards are pretty neat sometimes.',
});
```
